# 🚀 EduReach Platform

## 📌 Overview
EduReach ek full-stack e-learning platform hai jisme AI chat, authentication aur courses system hai.

## ✨ Features
- 🔐 User Authentication (Login/Signup)
- 🤖 AI Chat Support
- 📚 Course Listing
- 📞 Voice API Integration

## 🛠 Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB

## ▶️ Run Locally

### Client
cd edureach-platform/client
npm install
npm run dev

### Server
cd edureach-platform/server
npm install
npm run dev
